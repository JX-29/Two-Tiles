Two Tiles

How to play

This is a Two Tiles game. The gameplay is to find tiles with the same colors. The game is over when all the tiles have disappeared.
  • Press the Start button to start the game
  • If you need help click the Show Hints button
  • Select two tiles. If the tiles match colors, they will disappear. If the colors are different, the tiles will turn over
  • If you closed all tiles or want to start over, press the restart button
